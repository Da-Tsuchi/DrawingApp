### ペイントアプリ 概要
- 授業で制作したペイントアプリ

### 実装した機能
- #### フリーハンド、円、塗りつぶし円、四角、塗りつぶし四角、直線、全消去の描画モード
- #### 別ウィンドウで色の選択
- #### 円や四角の描写時に，途中経過を逐次描画し，マウスのボタンのリリース時に決定する
- #### 線の太さや破線等のプロパティを変更できる
- #### 消しゴム機能の実装


### 使用した技術
- Java

### 実行方法
- javaコンパイラをインストールすること
```
  $ cd DrawingApp
  $ javac DrawingApli00.java
  $ java DrawingApli00
```
